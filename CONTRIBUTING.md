# Contributing

Thanks for contributing.

## Development

1. Fork the repository.
2. Create a branch from `main`.
3. Make your changes.
4. Ensure the integration still validates in Home Assistant.
5. Open a pull request.

## Pull Request Checklist

- Keep changes focused and small.
- Update `CHANGELOG.md` when behavior changes.
- Add or update translations in `translations/*.json`.
- Verify `manifest.json` version is bumped for releasable changes.

## Scope

This project is an unofficial iSmartGate cloud integration for Home Assistant.
